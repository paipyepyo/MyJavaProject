# プログラム概要

NumberAnalyst.java は　ユーザーに数字を連続して入力してもらい、それぞれ数字の統計結果を返すクラスです。  
プログラムの流れは入力処理と表示処理処理は二つに分けられています。  

- **入力処理**：ユーザーに数字を入力してもらいます。
- **表示処理**：入力された数字を分析し、その数字の商や余り、符号、偶数か奇数などをコンソールに表示します。

## 入力ループ終了条件

入力中の数字を**すぐ前の数字と同じ入力**したら、入力処理の終了となります。続きまして、表示処理に進みます。  

## 表示する処理

入力されたそれぞれの数字の統計結果を、コンソールに行ごとに表示します。表示処理が終了すると、プログラムは終了します。　  

## 実行例
```text
入力をやめたい際は直前の数字と同じ入力してください
では、統計を分析するために、数字を一つずつ入力してください。

[ 1番目 ]   : 12.23
[ 2番目 ]   : -12.23
[ 3番目 ]   : -0.23
[ 4番目 ]   : 0.0
[ 5番目 ]   : 23.56
[ 6番目 ]   : 23.56


ご入力ありがとうございました。
それぞれの数字の統計結果は下記のとおりです。

[ 1番目 ]  : 数字：12.23  ,商：12  ,余り：23  ,符号：正   ,偶数か奇数：偶数
[ 2番目 ]  : 数字：-12.23 ,商：-12 ,余り：23  ,符号：負   ,偶数か奇数：偶数
[ 3番目 ]  : 数字：-0.23  ,商：0   ,余り：23  ,符号：負   ,偶数か奇数：偶数
[ 4番目 ]  : 数字：0.0    ,商：0   ,余り：0   ,符号：ゼロ ,偶数か奇数：偶数
[ 5番目 ]  : 数字：23.56  ,商：23  ,余り：56  ,符号：正   ,偶数か奇数：奇数
[ 6番目 ]  : 数字：23.56  ,商：23  ,余り：56  ,符号：正   ,偶数か奇数：奇数
```
