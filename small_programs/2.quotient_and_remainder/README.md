# プログラム概要

ShouuToAmari.java は　ユーザーに数字を連続して入力してもらい、それぞれ数字の商と余りを返すクラスです。  
プログラムの流れは入力処理と表示処理処理は二つに分けられています。  

- **入力処理**：ユーザーに数字を入力してもらいます。
- **表示処理**：入力された数字の商と余りをコンソールに表示します。

## 入力ループ終了条件

入力中の数字を**すぐ前の数字と同じ入力**したら、入力処理の終了となります。続きまして、表示処理に進みます。  

## 表示する処理

入力された数字のそれぞれをコンソールで行ごとに表示します。 表示処理終了したら、プログラムが終了します。　  

## 実行例
```text
入力をやめたい際は直前の数字と同じ入力してください
では商と余りを知りたい数字を一つずつ入力してください

[ 1番目 ] : 12.34
[ 2番目 ] : 56.78
[ 3番目 ] : 12.34
[ 4番目 ] : 12.34


ご入力ありがとうございました。
それぞれの商と余りは下記になります。

[ 1番目 ] : 数字：12.34, 商：12, 余り：34
[ 2番目 ] : 数字：56.78, 商：56, 余り：78
[ 3番目 ] : 数字：12.34, 商：12, 余り：34
[ 4番目 ] : 数字：12.34, 商：12, 余り：34
```
