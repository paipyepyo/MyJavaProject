# プログラム概要

Calculator.java は　計算機の基本的な機能をもっているプログラムです。  
主な五つの演算子で構成されています。
- 加算 (+)
- 減算 (-)
- 乗算 (*)
- 除算 (/)
- 剰余計算 (%)

# チェックして入力

人間の入力にも色々考えてプログラムが異常に終了しないように厳しく条件をチェックしています。  
計算用数字は**0.0入力禁止**、操作番号は**1～８**しか受け付けません。  
そして**操作番号は整数**で決まっているため、小数の入力も禁止です。  
また、計算機のプログラムなので**文字列はもちろん禁止**にしております。  

# サンプル入力・出力
```text
例 1：加算
計算機の操作番号(整数)を入力してください : 1
[ _ ] + [ x ] : 5
[ 5.0 ] + [ _ ] : 3
[ 5.0 ] + [ 3.0 ] = 8.0

例 2：整数除算（商のみ）
計算機の操作番号(整数)を入力してください : 4
[ _ ] / [ x ] : 7
[ 7.0 ] / [ _ ] : 2
[ 7.0 ] / [ 2.0 ] = 3

例 3：剰余計算
計算機の操作番号(整数)を入力してください : 6
[ _ ] % [ x ] : 7
[ 7.0 ] % [ _ ] : 3
[ 7.0 ] % [ 3.0 ] = 1
```
