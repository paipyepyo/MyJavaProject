# プログラム概要

Calculator.java は　計算機の基本的な機能をもっているプログラムです。  
主な五つの演算子で構成されています。
- 加算 (+)
- 減算 (-)
- 乗算 (*)
- 除算 (/)
- 剰余計算 (%)

# 4つの非機能要件を意識した設計・開発
このプログラムは4つの視点から考えて設計を行っております。4つの視点は以下通りになります。

## 1.安定性の視点
安定性の視点とは、正常時だけでなく、異常時においてもプログラムが停止せず動作し続けることを指します。  
本プログラムでは、ユーザー入力によるエラーを想定し、異常終了を防ぐために厳密な入力チェックを行っています。  
例えば、除算や剰余の場合はは**0.0入力禁止**、操作番号は**1～８**しか受け付けません。  
そして**操作番号は整数**で決まっているため、小数の入力も禁止です。  
また、計算機のプログラムなので**文字列はもちろん禁止**にしております。  <br> 
また、本プログラムにおける終了は以下の**2つ**に限定しています。<br> 
&nbsp;&nbsp;i.ユーザーの操作による**正常終了**（　操作番号8　）  
&nbsp;&nbsp;ii.シャットダウンや電源遮断などによる**強制終了**  

## 2.使う人の視点
使う人の視点とは、エンドユーザーにとって使いやすいプログラムを実現することです。<br>
どれほど多機能であっても、ユーザーにとって分かりにくいプログラムは価値が低いと考えています。<br><br> 
そのため、本プログラムではユーザーの操作性や視認性に配慮しています。<br> 
例えば、コンソール上で文字を整えて見やすく表示したり、操作番号によるメニューの再表示機能（操作番号7）を設けています。<br> 
また、無効な入力があった場合には、有効な入力内容を分かりやすく説明した上で再入力を促しています。<br> 
さらに、2番目の数値を入力する際には、直前に入力した1番目の数値も表示することで、現在の入力状況を把握しやすくしています。<br><br>
このように、ユーザーの立場に立った設計により、直感的で使いやすいプログラムを実現しています。<br>
