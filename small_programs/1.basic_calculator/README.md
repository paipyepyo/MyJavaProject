# プログラム概要

Calculator.java は　計算機の基本的な機能をもっているプログラムです。  
主な五つの演算子で構成されています。
- 加算 (+)
- 減算 (-)
- 乗算 (*)
- 除算 (/)
- 剰余計算 (%)

# 4つの非機能要件を意識した設計・開発
このプログラムは4つの視点から考えて設計を行っております。4つの視点は以下通りになります。

## 1.安定性の視点
安定性の視点とは、正常時だけでなく、異常時においてもプログラムが停止せず動作し続けることを指します。  
本プログラムでは、ユーザー入力によるエラーを想定し、異常終了を防ぐために厳密な入力チェックを行っています。  
例えば、除算や剰余の場合はは**0.0入力禁止**、操作番号は**1～８**しか受け付けません。  
そして**操作番号は整数**で決まっているため、小数の入力も禁止です。  
また、計算機のプログラムなので**文字列はもちろん禁止**にしております。  
また、本プログラムにおける終了は以下の2つに限定しています。 

&nbsp;&nbsp;i.ユーザーの操作による**正常終了**（操作番号8）  
&nbsp;&nbsp;ii.シャットダウンや電源遮断などによる**強制終了**
