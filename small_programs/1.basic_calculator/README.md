# プログラム概要

Calculator.java は　計算機の基本的な機能をもっているプログラムです。  
主な五つの演算子で構成されています。
- 加算 (+)
- 減算 (-)
- 乗算 (*)
- 除算 (/)
- 剰余計算 (%)

# 4つの非機能要件を意識した設計・開発
このプログラムは4つの視点から考えて設計を行っております。4つの視点は以下通りになります。

## 1.安定性の視点
安定性の視点とは正常時だけでなく、異常時にも停止しないで続くことです。  
当プログラムでは人間の入力にも色々考えてプログラムが異常に終了しないように厳しく条件をチェックしています。
